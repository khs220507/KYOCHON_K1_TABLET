# 설정 파일 안내

이 폴더에는 K1 시스템의 설정 파일들이 있습니다.
비개발자도 쉽게 수정할 수 있도록 JSON 형식으로 작성되어 있습니다.

## 파일 목록

### 1. `tcp_config.json` - TCP 통신 설정
로봇과 서버의 IP 주소 및 포트 번호를 설정합니다.

```json
{
  "robot": {
    "host": "0.0.0.0",        // 로봇 호스트 주소
    "port": 29999,             // 로봇 명령어 포트
    "feedbackPort": 30004      // 로봇 피드백 포트
  },
  "server": {
    "host": "localhost",       // 서버 호스트 주소 (IP 주소 또는 도메인)
    "port": 6601                // 서버 포트 (K1 시스템에서 메시지를 받는 포트)
  }
}
```

**수정 방법:**
- `server.host`: 서버의 IP 주소를 입력합니다
  - **Android 에뮬레이터**: `"10.0.2.2"` 사용 (호스트 컴퓨터를 가리킴)
  - **Android 실제 디바이스**: 컴퓨터의 IP 주소 사용 (예: `"192.168.1.100"`)
  - **Windows/Mac/Linux**: `"localhost"` 또는 `"127.0.0.1"` 사용
- `server.port`: 서버 포트 번호를 변경합니다
- `robot.host`, `robot.port`, `robot.feedbackPort`: 로봇 관련 설정 (일반적으로 변경 불필요)

### 2. `menu_config.json` - 메뉴 설정
각 메뉴의 조리 시간을 설정합니다.

```json
{
  "menus": [
    {
      "name": "후라이드 치킨",
      "preFryTime": 300,      // 초벌 시간 (초) - 초벌만 하는 시간
      "cookTime": 600,         // 조리 시간 (초) - 초벌 시간을 포함한 전체 조리 시간
      "shakeTime": 5,          // 흔들기 시간 (초)
      "shapeTime": 30          // 성형 시간 (초)
    }
  ]
}
```

**수정 방법:**
- `name`: 메뉴 이름을 입력합니다
- `preFryTime`: 초벌 시간을 초 단위로 입력합니다 (예: 300 = 5분)
- `cookTime`: 전체 조리 시간을 초 단위로 입력합니다 (초벌 시간 포함)
- `shakeTime`: 흔들기 시간을 초 단위로 입력합니다
- `shapeTime`: 성형 시간을 초 단위로 입력합니다

**새 메뉴 추가:**
- `menus` 배열에 새로운 메뉴 객체를 추가합니다
- 모든 필드(name, preFryTime, cookTime, shakeTime, shapeTime)를 입력해야 합니다

## 주의사항

1. **JSON 형식 준수**: 쉼표(,), 중괄호({}), 대괄호([])를 정확히 입력해야 합니다
2. **파일 저장 후 앱 재시작**: 설정 변경 후 앱을 재시작해야 적용됩니다
3. **백업 권장**: 설정 변경 전 원본 파일을 백업하는 것을 권장합니다
4. **숫자만 입력**: 시간 값은 숫자만 입력해야 합니다 (따옴표 없이)

## 예시

### 서버 IP 주소 변경

**Android 에뮬레이터인 경우:**
```json
{
  "server": {
    "host": "10.0.2.2",  // 에뮬레이터는 이 값 사용
    "port": 6601
  }
}
```

**Android 실제 디바이스인 경우:**
```json
{
  "server": {
    "host": "192.168.1.100",  // 컴퓨터의 실제 IP 주소로 변경
    "port": 6601
  }
}
```

**컴퓨터 IP 주소 확인 방법 (Windows):**
- 명령 프롬프트(cmd)에서 `ipconfig` 실행
- "IPv4 주소" 항목 확인 (예: 192.168.1.100)

**컴퓨터 IP 주소 확인 방법 (Mac/Linux):**
- 터미널에서 `ifconfig` 또는 `ip addr` 실행
- "inet" 항목 확인
